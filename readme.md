# 定时邮件发送

- superagent  发送网络请求
- cheerio  获取dom借点
- ejs模板引擎
- nodemailer 发送邮件插件
- node-schedule  定时器



思路步骤

1. 和风天气获取天气数据
2. 获取图片及文字
3. 模板引擎生成html
4. 定时器发送
